# The Search Atlas - Trending World

A website showcasing the most searched things in the world, with per-country trends, categorized trends, and interactive features.

## Features
- Global trending searches
- - Country-specific trends
  - - Category filtering
    - - Higher/Lower guessing game
      - - Newsletter subscription
        - - AdSense integration
          - - Stripe payments
